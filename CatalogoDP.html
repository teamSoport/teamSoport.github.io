Catálogo de Datos Personales del ITAIH

Catálogo de Datos Personales

Este Catálogo de Datos Personales es una herramienta para ayudar a los responsables del Instituto de Transparencia, Acceso a la Información Pública Gubernamental y Protección de Datos Personales del Estado de Hidalgo. Su objetivo es facilitar el manejo de datos personales y asegurar el cumplimiento de las leyes de transparencia y protección de datos.

¿Qué es un Catálogo de Datos Personales?

Un catálogo de datos personales es un documento que recopila información sobre los datos personales que una institución o entidad maneja. Este catálogo sirve como una herramienta para que las autoridades responsables de la protección de datos personales puedan identificar, clasificar y gestionar los datos que se recopilan. En otras palabras, es como un inventario que te dice qué tipo de información personal se está recopilando, para qué se utiliza y cómo se protege.
